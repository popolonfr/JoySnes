Ce patch permet de jouer à "Metal Gear 2 - Solid Snake" sur MSX avec
les 8 boutons d’une manette SNES grace à l'adaptateur JoySNES.


Le fichier “Metal Gear 2 [JoySNES].IPS” est le patch pour la version
japonaise de Metal Gear et les versions traduites.

Utiliser un des logiciels suivants pour appliquer le patch.
http://www.msx.org/wiki/How_to_use_IPS_files

Une fois le patch appliqué à la Rom, le jeu sera
jouable uniquement avec le JoySNES en mode Extra.

Utilisation des boutons :

 Croix de direction = Mouvements du joueur
 L = Ouvrir/Fermer le menu des armes
 R = Ouvrir/Fermer le menu de l’équipement
 X = Ramper (comme A + B)
 Y = Émetteur-récepteur
 A = Tirer/Sélectionner
 B = Coup de poing/Annuler/Passer les textes
 Start = Pause
 Select = Charger/Sauvegarder une partie

Le nom de fichier de sauvegarde s'écrit toujours au clavier.

Comment lancer le fichier Rom ?

- Si vous avez une MegaFlashRom au mapper konami SCC, utilisez le
  lanceur FL.COM (de GDX) ou OPFX.COM (de Manuel Pazos et Ramones).
- Si vous avez une MegaFlashRom SCC + SD, utilisez le lanceur
  OPFXSD.COM (de Manuel Pazos et Ramones).
- Si vous avez une MegaSram SCC, utilisez le lanceur MGLOAD.COM (de Tsujikawa)
  ou SROM.COM (de Louthrax).
- Si votre MSX a un Memory Mapper de 1024Ko ou plus et une cartouche SCC (sans
  ROM), essayez avec SROM.COM (de Louthrax) avec l'option /T. Attention,
  le jeu ne fonctionne pas avec la v2.1.


Ces solutions ne sont pas exhaustives.
