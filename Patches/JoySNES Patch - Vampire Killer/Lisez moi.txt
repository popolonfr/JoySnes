Ces patchs permettent de jouer à "Akumajou Dracula / Vampire Killer" sur MSX
avec une manette SNES à 8 boutons grace à l'adaptateur JoySNES.


Le fichier "VampKill [JoySNES].IPS" est le patch pour la ROM ayant
le crc32 : 5953d084.

Le fichier "VampKill-a [JoySNES].IPS" est le patch pour la ROM ayant
le crc32 : 0aedea40. Le patch peut être appliqué aussi sur la version
améliorée par le patch "Dynamic Vsync" par FRS.
http://frs.badcoffee.info/patches.html

Le fichier "VampKill-a2 [JoySNES].IPS" est le patch pour la ROM ayant
le crc32 : 038c250d.

Le fichier "VampKill-a3 [JoySNES].IPS" est le patch pour la ROM ayant
le crc32 : 217935d0. Le patch peut être appliqué aussi sur les versions
modifiées pour imiter Castlevania par Manuel Pazos & Imanok.
http://www.indieretronews.com/2016/06/vampire-killer-on-msx2-gets-castlevania.html
http://vk2cv.msxcartridgeshop.com/


Utiliser un des logiciels suivants pour appliquer le patch.
http://www.msx.org/wiki/How_to_use_IPS_files

Une fois le patch appliqué à la Rom, le jeu sera
jouable uniquement avec le JoySNES en mode Extra.

Utilisation des boutons :

 Croix de direction = Mouvements du joueur
 X = Utiliser le sablier / Rejouer au stage perdu (Game Master dans le slot 2)
 Y = Regarder le plan en possession
 L ou R = Lancer une fiole à gauche ou à droite
 A = Attaquer
 B = Sauter
 Start = Pause


Comment lancer le fichier Rom ?

- Si vous avez une MegaFlashRom au mapper ASCII 8k, utilisez le
  lanceur FL8.COM (de GDX).
- Si vous avez une MegaFlashRom au mapper konami sans SCC, utilisez le
  lanceur FLK4.COM (de GDX).
- Si vous avez une MegaFlashRom SCC, utilisez le lanceur FL.COM (de GDX)
  ou SROM.COM (de Louthrax).
- Si votre MSX a un Memory Mapper de 512Ko ou plus essayez avec
  SROM.COM (de Louthrax). Attention, le jeu ne fonctionne pas avec la v2.1.

Ces solutions ne sont pas exhaustives.
