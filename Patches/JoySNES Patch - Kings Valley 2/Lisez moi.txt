Ces patchs permettent de jouer à "King's Valley II - The Seal of El Giza" sur
MSX avec une manette SNES à 8 boutons grace à l'adaptateur JoySNES.


Le fichier "KVIIMSX1 [JoySNES].IPS" est le patch pour la version MSX1.

Le fichier "KVIIMSX2 [JoySNES].IPS" est le patch pour la version MSX2.
 
Le fichier "KVIIECYS [JoySNES].IPS" est le patch pour la version
"Edit Contest Yuushuu Sakuhinshuu".

Utiliser un des logiciels suivants pour appliquer le patch.
http://www.msx.org/wiki/How_to_use_IPS_files

Une fois le patch appliqué à la Rom, le jeu sera
jouable uniquement avec le JoySNES en mode Extra.


Utilisation des boutons pendant le jeu :

 Croix de direction = Mouvements du joueur
 A = Utiliser un objet
 B = Sauter
 Start = Pause
 Select = Abandonner
 Y = Affiche le plan du stage pendant la pause
 X = Jouer / Couper la musique

Utilisation des boutons en mode édition :

 Droite et Gauche = Déplacement du curseur (nom de fichier / mot de passe)
 X = Annuler / Rejouer le stage
 Y = OK
 L = Passer de l'éditeur au menu des options et vice versa / Charger un fichier


Comment lancer le fichier Rom ?

- Si vous avez une MegaFlashRom au mapper konami SCC, utilisez le
  lanceur FL.COM (de GDX) ou OPFX.COM (de Manuel Pazos et Ramones).
- Si vous avez une MegaFlashRom SCC + SD, utilisez le lanceur
  OPFXSD.COM (de Manuel Pazos et Ramones).
- Si vous avez une MegaSram SCC, utilisez le lanceur MGLOAD.COM (de Tsujikawa)
  ou SROM.COM (de Louthrax).
- Si votre MSX a un Memory Mapper de 256Ko ou plus et une cartouche SCC (sans
  ROM), essayez avec SROM.COM (de Louthrax) avec l'option /T. Attention, le
  jeu ne fonctionne pas avec la v2.1.

Ces solutions ne sont pas exhaustives.
